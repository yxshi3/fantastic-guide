I have received multiple compliments and requests to publish the current design of my website as a theme. I recognized that it is often hard when you are trying to code your first personal site because you are overwhelmed by the number of things that you need to consider and don't know where to start. So I hope by releasing this theme on GitHub with inline code comments, more people could find coding their first personal site or portfolio easier.

After taking the web.lab in winter 2021, I decided to apply HTML, CSS, JavaScript, and UI/UX knowledge that I learned by redesigning my old personal website. In comparison to version 1.0, I wrote all of my code instead of using a bootstrap framework. Coding my website from scratch has helped me to have a better understanding of web development and to improve my programming skills in terms of readability. If you found a bug or have a feature request, feel free to submit your issues.

I'm working hard to make this theme available as soon as possible. Check back soon or view my current progress!
